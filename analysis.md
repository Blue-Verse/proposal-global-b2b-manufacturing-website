## 2. URL/이미지 분석

### 레퍼런스 A — 현대코퍼레이션 (http://www.hyundaicorp.com/)
- WebFetch 접속 실패 (서버 측 연결 차단, HTTP-only). 공고 설명 기준으로 반영.
- 톤앤매너: 와이드한 풀스크린 비주얼 + 넉넉한 여백을 활용한 글로벌 대기업형 레이아웃.
- 반영: 메인 히어로/서브 페이지 공통 그리드를 와이드 비주얼 + 여백 중심으로 설계.

### 레퍼런스 B — 씨노스 / LUCINOVA (https://www.cinos21.com/)
- WebFetch 분석 완료.
- 내비게이션: 회사정보 / R&D / 제품 포트폴리오 / 신사업 / ESG / Contact의 계층 메뉴.
- 제품 카탈로그: 카테고리(Ambient Light, Indicators 등)별 분류 + "Read more" → 상세 사양 페이지 계층 구조.
- 디자인: 미니멀, 단색 브랜딩(다크/라이트 로고), 생산 시설 사진, 넉넉한 여백.
- 기능: **한국어·영어·일본어 다국어**, 회사명·연락처·사양을 받는 정형 문의 폼, 개인정보 수집 고지, 글로벌 거점 맵(한국·중국·인도), 클라이언트 로고·인증 갤러리.
- 반영: 제품 카탈로그 계층 IA, 다국어 구조, 정형 Contact Us 폼 패턴을 본 프로젝트 IA로 재구성.

## 3. 실현 가능성 분석 (내부용)

- 프로젝트 유형: FE 중심 기업 사이트(제품 카탈로그) → **강력 추천** 영역 (+0%~+10% 버퍼, 다국어로 소폭 가산)
- 기본 공수 (효율 미반영): 약 47 M/D
- 조정 공수 (효율 반영 + 다국어 버퍼): 약 28 M/D
  - 기획 6×0.65=3.9 / 디자인 12×0.85=10.2 / FE 15×0.35=5.25 / BE 8×0.45=3.6 / 인프라 2×0.35=0.7 / QA 4×0.45=1.8 ≈ 25.5 → 버퍼 +10% ≈ 28 M/D
- 달력 일수 (1인): 28 × 7/5 ≈ 39일
- 클라이언트 예상 기간: 60일
- 판정: **60일 내 충분히 가능** — 다수 인력 투입으로 단계 병행, 기간 60일 유지. 여유로 다국어 검수·인프라 대행 안정적 수용.
- 공수계산서(클라이언트용)는 실제 작업량 기준 56 M/D로 제시 (다수 인력 60일 배분).
