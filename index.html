핵심 요구사항 분석

  • 글로벌 타깃 B2B 제조·기술 기업 웹사이트 순수 신규 구축 (리뉴얼 아님) — 기획·UI/UX 디자인·퍼블리싱·개발 전 과정
  • 국문 기준 약 15~20 페이지 규모, 한국어·영어·일본어 3개 국어 다국어 확장 필수
  • PC 전용 환경 최적화 (모바일·태블릿 반응형 제외, 데스크톱 해상도 고정)
  • B2B 기업 소개 + 제품 카탈로그 노출 중심 구성, 다국어 이메일 문의 폼(Contact Us) 연동
  • 초기 웹 호스팅 세팅 대행 + 도메인 2종(.com 영문 / 한글) 구매·관리 및 메인 도메인 통합 포워딩 세팅 대행
  • 커머스·비즈매칭·회원가입/로그인 등 특수 시스템 없음 (기본 웹 프로그램 위주)
  • 데이터 마이그레이션 해당 없음 (신규 구축)
  • 글로벌 시장 신뢰감 확보를 위한 프리미엄 2D 미니멀 레이아웃 지향

제안 포인트

1. 두 레퍼런스의 강점을 결합한 설계 — 분석 완료

  • 현대코퍼레이션(톤앤매너): 와이드한 풀스크린 비주얼과 넉넉한 여백을 활용한 글로벌 대기업형 레이아웃을 메인/서브 공통 그리드로 구조화하여 신뢰감 있는 첫인상을 설계합니다.
  • 씨노스(구조·기능): 참고 사이트 분석 결과, 제품을 카테고리별 계층으로 정리하는 카탈로그 구조, 글로벌 거점 맵, 회사명·연락처·사양을 받는 정형 문의 폼 패턴이 확인되었습니다. 해당 IA를 본 프로젝트에 맞게 재구성하여 직관적인 제품 노출 동선을 구현합니다.

2. 다국어(한·영·일) 확장 구조 우선 설계

  • i18n 기반 다국어 라우팅으로 콘텐츠를 분리 관리하여, 추후 언어 추가·수정 시에도 운영 부담을 최소화합니다.
  • 언어별 SEO 메타·hreflang·OG 태그 적용으로 글로벌 검색 노출에 유리한 구조를 확보합니다.

3. 인프라·도메인 원스톱 대행

  • 웹 호스팅 초기 세팅, 영문(.com)·한글 도메인 2종 구매 및 메인 도메인 통합 포워딩까지 일괄 대행하여 클라이언트의 기술적 부담을 제거합니다.

4. 운영 편의를 위한 콘텐츠 관리(CMS) 제공

  • 제품 카탈로그·게시판 콘텐츠를 비개발자도 직접 등록·수정할 수 있는 관리 환경을 제공하여 오픈 이후의 자생적 운영을 지원합니다.

PRD (Product Requirements Document)

프로젝트 요구사항을 정의하고 기능 범위를 명확히 합니다.

1. 프로젝트 배경 및 목적

글로벌 B2B 제조·기술 기업의 웹사이트를 순수 신규로 구축합니다. 글로벌 시장에서의 신뢰감 확보와 직관적인 제품 노출을 핵심 목표로 하며, 한·영·일 3개 국어를 지원하여 해외 잠재 고객과의 접점을 확대합니다. 금년 내 최종 오픈 및 사이트 활성화를 목표로 합니다.

2. 사용자 정의

사용자 유형설명주요 기능
해외/국내 잠재 고객 (B2B)제품·기술을 탐색하는 글로벌 바이어·파트너다국어 기업 소개 열람, 제품 카탈로그 탐색, 문의 폼 발송
일반 방문자브랜드·기업 정보를 확인하는 방문자회사 소개, 연혁, 뉴스/게시판 열람
사이트 관리자콘텐츠를 운영·관리하는 클라이언트 담당자제품·게시글 등록/수정, 다국어 콘텐츠 관리, 문의 내역 확인

3. 기능 요구사항

3.1 기업 소개 (Company)

  • [필수] 회사 개요, 비전/미션, 연혁, 글로벌 거점/조직 등 기업 소개 페이지군
  • [필수] 와이드 비주얼·여백 중심의 글로벌 대기업형 메인 레이아웃

3.2 제품 카탈로그 (Products)

  • [필수] 제품/기술 카테고리 계층 구조 및 상세 페이지
  • [필수] 관리자에 의한 제품 등록/수정/노출 관리(CMS)

3.3 다국어 (i18n)

  • [필수] 한국어·영어·일본어 3개 국어 전체 페이지 지원 및 언어 전환
  • [필수] 언어별 SEO 메타·hreflang 태그 적용

3.4 문의 (Contact Us)

  • [필수] 다국어 이메일 문의 폼 (회사명·연락처·문의 내용 등), 메일 발송 연동
  • [필수] 개인정보 수집·이용 동의 및 스팸 방지(캡차)

3.5 게시판 (Board)

  • [선택] 뉴스/공지 등 게시판 기능 및 관리자 작성

5. 기술 제약사항

  • PC 전용 고정 레이아웃 — 반응형은 범위에서 제외
  • 도메인 2종(.com 영문 / 한글)을 메인 도메인으로 통합 연결하는 포워딩 세팅 필요
  • 기존 사이트·DB 없음 → 데이터 마이그레이션 해당 없음

6. 범위 외 사항 (Out of Scope)

  • 쇼핑몰(커머스)·비즈 매칭 시스템
  • 회원가입/로그인 등 별도 회원 시스템
  • 회원 정보 이관·기존 게시글 마이그레이션
  • 모바일·태블릿 반응형 대응
